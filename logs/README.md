# Logs Directory

本ディレクトリには、実行ログファイルが格納されます。

## ログファイルの種類

### 評価ログ

評価スクリプトの実行ログ:
- `evaluation_*.log` - 評価実行の詳細ログ
- タイムスタンプ、エラーメッセージ、警告などが記録されます

### サービスログ

長時間実行されるサービスのログ:
- `ollama.log` - Ollama サービスのログ（setup/ にも存在）
- その他のバックグラウンドサービスログ

## ログローテーション

ログファイルが大きくなった場合は手動で削除してください:

```bash
# 古いログの削除
rm logs/*.log

# または特定のパターン
rm logs/evaluation_20251105_*.log
```

## 注意事項

- すべてのログファイル (`.log`) は Git 管理から除外されています
- ログファイルは自動的にローテーションされません
- ディスク容量に注意してください
